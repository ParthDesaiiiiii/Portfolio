A professional, modern portfolio website showcasing my work as a Software Engineer, Full Stack Developer, and Web Developer.

- **Modern React Architecture**: Built with React 18 and functional components

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository or navigate to the project directory:
   ```bash
   cd Portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Building for Production

To create a production build:

```bash
npm run build
```

This builds the app for production to the `build` folder. The build is optimized and ready to be deployed.

## Customization

To customize the portfolio with your own information:

1. Update personal information in each component file
2. Modify colors in `src/index.css` (CSS variables)
3. Update social media links in `Hero.js`, `Contact.js`, and `Footer.js`
4. Add or remove projects in `Projects.js`
5. Update experience and education details in respective component files

## Deployment

This portfolio can be deployed to various platforms:

- **Netlify**: Drag and drop the `build` folder
- **Vercel**: Connect your GitHub repository
- **GitHub Pages**: Use `gh-pages` package
- **AWS S3**: Upload the `build` folder
